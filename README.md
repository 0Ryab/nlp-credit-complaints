# NLP Credit Complaints

Sistema de análise de reclamações bancárias usando NLP e IA Generativa.

## Problema de Negócio
Instituições financeiras recebem milhares de reclamações diariamente.
Classificar manualmente o sentimento e urgência de cada uma é inviável
em escala. Este projeto automatiza essa análise usando modelos de linguagem
modernos e cria uma base de conhecimento inteligente para suporte ao cliente.

## Tecnologias
- Python 3.13
- Hugging Face Transformers / BERT
- LangChain
- ChromaDB
- Streamlit
- Pytest
