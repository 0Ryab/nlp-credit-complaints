# NLP Credit Complaints

Sistema de análise de reclamações bancárias com NLP e IA Generativa,
desenvolvido como case técnico para vaga de Cientista de Dados com foco
em NLP e IA Generativa no setor financeiro.

## Problema de Negócio
Instituições financeiras recebem milhares de reclamações diariamente.
Classificar manualmente o sentimento e urgência de cada uma é inviável
em escala. Este projeto automatiza essa análise usando modelos de linguagem
modernos.

## O que o projeto entrega
- Classificador de sentimentos de reclamações bancárias com BERT
- Sistema RAG para responder perguntas sobre produtos de crédito
- Interface conversacional via Streamlit

## Tecnologias
- Python 3.13
- Hugging Face Transformers
- BERT
- LangChain
- ChromaDB (base vetorial)
- Streamlit
- Pytest
